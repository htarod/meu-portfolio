Regressão Linear: ENEM x Número de Matrículas (SP)

Neste projeto, apliquei regressão linear simples para verificar se existe relação entre o número de matrículas e o desempenho médio das escolas públicas no ENEM de 2015 no estado de São Paulo.

Em outras palavras: tentei descobrir se escolas com mais alunos vão melhor ou pior no ENEM — usando matemática para entender essa possível relação.

Fonte de Dados

Os dados foram extraídos do conjunto "Microdados ENEM por Escola 2015". Após filtragem, mantive apenas escolas públicas do estado de São Paulo.

Etapas realizadas

  • Limpeza e filtragem dos dados com pandas
  • Visualização com seaborn e matplotlib
  • Ajuste do modelo de regressão com statsmodels
  • Cálculo do R², erro padrão e coeficientes

Visualização

Resultados

O modelo indicou correlação fraca entre número de matrículas e nota média. O R² foi inferior a 0.1, o que sugere que outros fatores são mais relevantes para o desempenho escolar.

Reflexão

A baixa correlação reforça a importância de aprofundar a análise com variáveis como investimento por aluno, infraestrutura, localização e perfil socioeconômico. Esse projeto mostra como a estatística pode indicar quando não há relação significativa.
