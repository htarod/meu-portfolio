Controle de Estoque com Excel e VBA

Neste projeto, desenvolvi uma ferramenta simples e funcional para controlar o estoque de produtos diretamente no Excel, utilizando VBA. O objetivo foi criar uma solução acessível para quem precisa registrar entradas e saídas com rapidez, sem depender de sistemas externos.

Em outras palavras: é uma planilha inteligente que registra tudo o que entra e sai do estoque, mostra o saldo atual e avisa quando um produto está acabando.

Funcionalidades

  • Formulário intuitivo para registrar movimentações
  • Planilha com controle por categorias
  • Alerta automático para estoques baixos
  • Histórico de todas as movimentações
  • Botão de atalho na aba inicial

Aplicação

Empresas ou pessoas que precisam controlar manualmente o estoque podem utilizar esta ferramenta. Basta abrir a planilha, clicar no botão da aba "Início" e preencher o formulário.

Fórmula de Alerta

Na coluna "Alerta" da aba de estoque, utilizei esta fórmula para indicar quando um produto está abaixo do mínimo definido: 

=SE(D2<E2; "Estoque Baixo"; "Normal")

Trecho de Código VBA

Este exemplo mostra como uma entrada é registrada automaticamente: 

If tipo = "Entrada" Then
    wsEstoque.Cells(linhaProduto, 4).Value = wsEstoque.Cells(linhaProduto, 4).Value + qtde
    ultimaLinha = wsEntradas.Cells(wsEntradas.Rows.Count, 1).End(xlUp).Row + 1
    wsEntradas.Cells(ultimaLinha, 1).Value = Date
    wsEntradas.Cells(ultimaLinha, 2).Value = codigo
    wsEntradas.Cells(ultimaLinha, 3).Value = produto
    wsEntradas.Cells(ultimaLinha, 4).Value = qtde
    wsEntradas.Cells(ultimaLinha, 5).Value = operador
End If
