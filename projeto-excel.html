Projeto com Excel Avançado

Descrição: Neste projeto, criei dashboards dinâmicos com fórmulas avançadas, validações, macros e Power Query para automatizar a análise de dados de uma empresa fictícia.

Objetivo: Demonstrar domínio de recursos avançados do Excel aplicados a controle de vendas e produtividade de equipes.

Tecnologias utilizadas: Excel, Power Query, Tabelas Dinâmicas, VBA básico.

Etapas do processo:

  • Importação e limpeza de dados com Power Query
  • Uso de fórmulas como PROCV/XLOOKUP, ÍNDICE + CORRESP, SOMASES
  • Criação de painéis com gráficos interativos e segmentações
  • Automatização com VBA simples (botões para limpar filtros e atualizar dados)

Resultado: Painel gerencial claro e funcional para visualização rápida de metas, desempenho mensal e análise por setor.
