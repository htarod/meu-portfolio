Projeto de Processamento de Linguagem Natural (NLP)

Descrição: Desenvolvi uma aplicação de análise de sentimentos usando dados de avaliações de clientes em português, com foco em identificar padrões e insights.

Objetivo: Criar um pipeline de NLP capaz de classificar sentimentos (positivo, negativo, neutro) com visualização dos resultados.

Tecnologias utilizadas: Python, NLTK, scikit-learn, pandas, matplotlib, Jupyter.

Etapas do processo:

  • Pré-processamento dos textos (limpeza, tokenização, stopwords)
  • Extração de features com TF-IDF
  • Treinamento de modelos de classificação (Logistic Regression e SVM)
  • Avaliação do desempenho com métricas e visualização de confusão

Resultado: Acurácia de mais de 85% com dashboard simples de análise de sentimentos por palavra-chave.
