Dashboard de Indicadores Econômicos com Power BI

Neste projeto, desenvolvi um dashboard interativo no Power BI utilizando dados do IPCA (Índice de Preços ao Consumidor Amplo), extraídos do site do Banco Central e organizados com Python e SQL.

Em outras palavras: criei um painel visual que mostra a inflação no Brasil, com gráficos interativos construídos a partir de dados reais e atualizados automaticamente.

Objetivo

O foco foi mostrar a evolução da inflação nos últimos anos, permitindo que qualquer pessoa consiga navegar pelos dados, entender médias mensais, tendências, variações por estado e comparar indicadores.

Visualizações

O dashboard traz:

  • Evolução histórica do IPCA
  • Variação mensal e acumulada
  • Média e desvio padrão por grupo de produtos
  • Segmentações por ano, categoria e região

Como os dados foram tratados

Antes de importar para o Power BI, usei Python para coletar os dados do Banco Central via API. Em seguida, salvei os dados em uma base SQLite e apliquei consultas SQL para gerar métricas como média, desvio padrão e percentuais de variação.

Por que Power BI?

Escolhi o Power BI por ser uma das ferramentas mais utilizadas em empresas para análise visual e relatórios executivos. Além disso, ele permite atualizações automáticas a partir de arquivos locais ou APIs externas.
