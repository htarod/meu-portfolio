SQL e Power BI: Automação na Análise de Indicadores Econômicos

Neste projeto, combinei Python, SQL e Power BI para automatizar a coleta e visualização de indicadores econômicos. O foco foi o IPCA, mas a estrutura pode ser aplicada a qualquer conjunto de dados que utilize séries temporais.

Em outras palavras: conectei Python, banco de dados e Power BI para montar um painel dinâmico que se atualiza sozinho com novos dados econômicos.

Etapas do projeto

  • Extração dos dados via API do Banco Central
  • Processamento e limpeza com Python
  • Armazenamento em SQLite
  • Consulta com SQL para gerar métricas
  • Conexão do banco de dados local com Power BI

Consultas SQL utilizadas


SELECT strftime('%Y-%m', data) AS mes, AVG(valor) AS media
FROM ipca
GROUP BY mes;

Muitas empresas ainda dependem de arquivos locais. Com essa solução, mostro que é possível automatizar a análise mesmo sem servidores remotos ou ferramentas pagas.
