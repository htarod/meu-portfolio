Projeto: Análise de Vendas com ETL

Descrição: Neste projeto, realizei uma análise completa do desempenho de vendas de uma empresa fictícia. A partir de um conjunto de dados brutos, apliquei técnicas de ETL (Extração, Transformação e Carga) para preparar os dados, e em seguida gerei insights por meio de gráficos e estatísticas descritivas.

Etapas:

  • Coleta dos dados em formato CSV
  • Limpeza e transformação com Python (Pandas)
  • Criação de gráficos e tabelas com Matplotlib e Seaborn
  • Interpretação de métricas relevantes

Ferramentas utilizadas: Python, Pandas, Matplotlib, VS Code, GitHub
