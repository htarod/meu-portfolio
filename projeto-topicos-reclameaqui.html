Análise de Tópicos em Reclamações do Reclame Aqui

Este projeto tem como objetivo aplicar técnicas de Processamento de Linguagem Natural (NLP) e modelagem de tópicos para analisar reclamações de consumidores extraídas do site Reclame Aqui. A ideia é identificar automaticamente os principais assuntos abordados nas reclamações, agrupando os textos em tópicos coerentes.

Etapas do Projeto

  • Coleta de dados: Utilização de um dataset com milhares de reclamações públicas da plataforma Reclame Aqui.
  • Pré-processamento: Limpeza dos textos, remoção de stopwords em português e tokenização.
  • Modelagem de Tópicos (LDA): Aplicação do modelo LDA (Latent Dirichlet Allocation) para extrair os principais temas tratados nas reclamações.
  • Visualização: Geração de visualizações gráficas, incluindo uma visualização interativa com a biblioteca PyLDAvis.

Aplicações Práticas

Essa análise pode ser útil para empresas que desejam:

  • Identificar os principais problemas relatados pelos clientes.
  • Monitorar a reputação da marca.
  • Priorizar melhorias nos seus serviços ou produtos.
  • Automatizar a categorização de reclamações para atendimento mais eficiente.

Dataset Utilizado

O dataset original foi processado para incluir somente colunas relevantes como título e texto completo das reclamações. O arquivo utilizado foi reclamacoes_completas.csv, contendo milhares de registros.

Tecnologias e Bibliotecas

  • Python 3.10
  • Pandas, NumPy, Matplotlib, Seaborn
  • NLTK, Gensim, Scikit-learn
  • PyLDAvis para visualização interativa
